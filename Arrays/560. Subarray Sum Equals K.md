# 560. Subarray Sum Equals K

## Pattern
Prefix Sum + HashMap

## Recognition
- Count subarrays with a given sum.
- Array may contain negative numbers.
- Sliding Window doesn't work because the window sum is not monotonic.

## Core Idea
Let:

prefixSum = sum of elements from index 0 to i

For a subarray ending at index `i` to have sum `k`:

```text
prefixSum - previousPrefix = k

previousPrefix = prefixSum - k
```

Store the frequency of every prefix sum in a HashMap.

## Algorithm
1. Initialize `sum = 0`, `count = 0`.
2. Store `(0,1)` in HashMap.
3. Traverse the array:
   - Add current element to `sum`.
   - If `(sum - k)` exists in map:
     - Add its frequency to answer.
   - Store/update current prefix sum in map.
4. Return count.

## Pseudocode

```java
sum = 0
count = 0

map.put(0, 1)

for(num : nums):

    sum += num

    if(map contains (sum - k)):
        count += map.get(sum - k)

    map.put(sum, map.getOrDefault(sum, 0) + 1)

return count
```

## Key Syntax

```java
HashMap<Integer, Integer> map = new HashMap<>();

map.put(0, 1);

sum += nums[i];

map.containsKey(sum - k);

count += map.get(sum - k);

map.put(sum, map.getOrDefault(sum, 0) + 1);
```

## Complexity

Time: O(n)

Space: O(n)

## Template Insight

Whenever:
- Count subarrays.
- Given target sum.
- Negative numbers present.

Think:
**Prefix Sum + HashMap**

> If only positive numbers are present, Sliding Window may work instead.
