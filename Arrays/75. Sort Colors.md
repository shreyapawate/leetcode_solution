# 75. Sort Colors

## Pattern
Dutch National Flag Algorithm (Three Pointers)

## Recognition
- Array contains only 0, 1, 2.
- Sort in one pass.
- O(1) extra space expected.

## Core Idea
Maintain three regions:

- [0 ... low-1] → 0s
- [low ... mid-1] → 1s
- [mid ... high] → Unknown
- [high+1 ... n-1] → 2s

Process the unknown region using `mid`.

## Algorithm
1. low = 0, mid = 0, high = n - 1
2. While mid <= high:
   - If nums[mid] == 0:
     - swap(low, mid)
     - low++, mid++
   - If nums[mid] == 1:
     - mid++
   - If nums[mid] == 2:
     - swap(mid, high)
     - high--
     - Do NOT increment mid

## Complexity
Time: O(n)
Space: O(1)

## Template Insight
Whenever:
- Array contains only 3 categories.
- Need partitioning around multiple values.
- In-place sorting required.

Think Dutch National Flag.

## Common Mistakes
- Incrementing mid after swapping with high.
- Using a sorting algorithm (O(n log n)).
- Incorrect loop condition (`mid <= high`).
- Forgetting that swapped element from high is unprocessed.
