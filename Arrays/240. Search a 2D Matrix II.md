# 240. Search a 2D Matrix II

## Pattern
Matrix Search (Staircase Search)

## Recognition
- Rows are sorted.
- Columns are sorted.
- Matrix is **not** globally sorted.
- Need efficient search without checking every element.

## Core Idea
Start from the **top-right** corner.

At any position:
- Left elements are **smaller**.
- Bottom elements are **larger**.

So:
- If current > target → move **left**.
- If current < target → move **down**.
- If current == target → found.

Each move eliminates one row or one column.

## Algorithm
1. Start at `(0, cols - 1)`.
2. While inside the matrix:
   - If current == target → return true.
   - If current > target → move left.
   - Else → move down.
3. If search ends → return false.

## Pseudocode

```java
row = 0
col = cols - 1

while(row < rows && col >= 0):

    if(matrix[row][col] == target)
        return true

    if(matrix[row][col] > target)
        col--
    else
        row++

return false
```

## Key Syntax

```java
int row = 0;
int col = matrix[0].length - 1;

while(row < matrix.length && col >= 0)

matrix[row][col]

row++;

col--;
```

## Complexity

Time: O(m + n)

Space: O(1)

## Template Insight

Whenever:
- Every row is sorted.
- Every column is sorted.
- Need to search efficiently.

Think:
**Start at Top-Right (or Bottom-Left)**

Reason:
- One move always eliminates an entire row or column.
