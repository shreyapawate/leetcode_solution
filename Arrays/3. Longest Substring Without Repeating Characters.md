# 3. Longest Substring Without Repeating Characters

## Pattern
Sliding Window + HashSet

## Recognition
- Longest/maximum subarray or substring.
- Need unique/distinct characters.
- Window needs to expand and shrink dynamically.

## Core Idea
Maintain a window containing unique characters.

- Expand right pointer.
- If duplicate appears, shrink from left until duplicate is removed.
- Track maximum window length.

## Algorithm
1. Initialize left = 0.
2. Traverse string using right pointer.
3. If current character already exists:
   - Remove characters from left until duplicate is removed.
4. Add current character.
5. Update maximum length.
6. Return maximum length.

## Pseudocode

```java
left = 0
maxLen = 0
set = {}

for(right = 0 to n-1):

    while(set contains s[right]):
        set.remove(s[left])
        left++

    set.add(s[right])

    maxLen = max(maxLen, right - left + 1)

return maxLen
```

## Key Syntax

```java
HashSet<Character> set = new HashSet<>();

set.add(ch);

set.contains(ch);

set.remove(ch);

Math.max(maxLen, right - left + 1);
```

## Complexity

Time: O(n)
Space: O(min(n, charset))

## Template Insight

For variable-size window problems:

```text
Expand right
Violation occurs
Shrink left
Update answer
```

This is one of the most important Sliding Window templates.
