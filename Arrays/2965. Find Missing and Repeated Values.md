class Solution {
    public int[] findMissingAndRepeatedValues(int[][] grid) {
        int r=grid.length;
        int c=grid[0].length;
      
        int[]hash= new int[r*c+1];
        for(int i=0;i<r;i++){
            for(int j=0;j<c;j++){
                hash[grid[i][j]]++;
            }
        }
        int[]ans= new int[2];
        for(int i=1;i<=r*c;i++){
            if(hash[i]==2){
                ans[0]=i;
            }
            if(hash[i]==0){
                ans[1]=i;
            }
        }
         return ans;
    }
   
}
# 2965. Find Missing and Repeated Values

## Pattern
Frequency Counting (Hashing)

## Recognition
- Numbers are from 1 to n².
- One number is repeated.
- One number is missing.
- Need to identify both.

## Core Idea
Count the frequency of every number.

- Frequency = 2 → Repeated number.
- Frequency = 0 → Missing number.

## Algorithm
1. Create frequency array of size n² + 1.
2. Traverse grid and increment frequency.
3. Traverse 1 to n²:
   - freq[i] == 2 → repeated
   - freq[i] == 0 → missing
4. Return [repeated, missing].

## Complexity
Time: O(n²)
Space: O(n²)

## Template Insight
Whenever:
- Elements belong to a known range.
- Need duplicate/missing detection.

Think Frequency Array before HashMap.

## Common Mistakes
- Using n = rows + cols instead of n².
- Starting traversal from wrong index.
- Forgetting frequency array size should be n² + 1.
- Returning [missing, repeated] instead of [repeated, missing].
