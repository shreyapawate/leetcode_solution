# 54. Spiral Matrix

## Pattern
Matrix Traversal (Boundary Simulation)

## Recognition
- Traverse matrix in spiral order.
- Need to visit every element exactly once.
- Four-direction traversal with shrinking boundaries.

## Core Idea
Maintain four boundaries:

- top
- bottom
- left
- right

Traverse in order:
1. Left → Right
2. Top → Bottom
3. Right → Left
4. Bottom → Top

After each traversal, shrink the corresponding boundary.

## Algorithm
1. Initialize `top`, `bottom`, `left`, `right`.
2. While `top <= bottom` and `left <= right`:
   - Traverse top row.
   - Increment `top`.
   - Traverse right column.
   - Decrement `right`.
   - If rows remain:
     - Traverse bottom row.
     - Decrement `bottom`.
   - If columns remain:
     - Traverse left column.
     - Increment `left`.
3. Return answer.

## Pseudocode

```java
top = 0
bottom = rows - 1
left = 0
right = cols - 1

while(top <= bottom && left <= right):

    // Left → Right
    for(i = left; i <= right; i++)
        ans.add(matrix[top][i])
    top++

    // Top → Bottom
    for(i = top; i <= bottom; i++)
        ans.add(matrix[i][right])
    right--

    // Right → Left
    if(top <= bottom):
        for(i = right; i >= left; i--)
            ans.add(matrix[bottom][i])
        bottom--

    // Bottom → Top
    if(left <= right):
        for(i = bottom; i >= top; i--)
            ans.add(matrix[i][left])
        left++

return ans
```

## Complexity

Time: O(m × n)

Space: O(1) extra (excluding output list)
