# 151. Reverse Words in a String

## Pattern
String Manipulation + Two Pointers

## Recognition
- Reverse the order of words.
- Remove leading/trailing spaces.
- Reduce multiple spaces to a single space.
- Preserve characters within each word.

## Core Idea
Reverse the entire string, then reverse each individual word.

Steps:
1. Trim extra spaces.
2. Reverse the whole string.
3. Reverse every word.
4. Build the final string with single spaces.

> **Alternative:** Split the string into words and append them in reverse order (uses extra space).

## Algorithm
1. Remove extra spaces.
2. Reverse the entire string.
3. Traverse the string:
   - Find each word.
   - Reverse that word.
4. Return the processed string.

## Pseudocode

```java
removeExtraSpaces()

reverse(0, n-1)

start = 0

for(end = 0 to n):

    if(end == n || s[end] == ' '):
        reverse(start, end-1)
        start = end + 1

return string
```

## Key Syntax

```java
StringBuilder sb = new StringBuilder();

sb.reverse();

sb.append(ch);

sb.toString();

s.trim();

s.split("\\s+");      // Alternative approach
```

## Complexity

### Reverse-in-place (using StringBuilder/char array)
Time: O(n)

Space: O(1) extra (excluding mutable string storage)

### Split approach
Time: O(n)

Space: O(n)

## Template Insight

Whenever:
- Reverse word order.
- Normalize spaces.
- Reverse words but not characters.

Think:
**Reverse Whole → Reverse Each Word**
